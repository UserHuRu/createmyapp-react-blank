# createmyapp-react-blank

> react app skeleton based on create-react-app

### Feature
* Based on create-react-app & react-app-rewire
* React-Router
* Redux & Redux Persist
* Ant Design

### Quick Start

```bash
$ yarn i -g create-my-app
$ create-my-app init react-blank <project name>
$ cd <project name>
$ yarn
$ yarn start
```
